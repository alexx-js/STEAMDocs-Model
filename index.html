¿Que es HTML?


HTML (HyperText Markup Language) es un lenguaje de marcado utilizado para crear páginas web. Permite estructurar y dar formato al contenido de una página web mediante el uso de etiquetas y atributos, que describen la apariencia y el comportamiento de los elementos en la página. HTML es el lenguaje base de la web y es interpretado por los navegadores web para mostrar el contenido de una página a los usuarios.


Estructura de un documento HTML


La estructura básica de un documento HTML consta de varias partes principales que se definen mediante etiquetas. A continuación, se describe la estructura básica de un documento HTML:

Etiqueta HTML: El documento HTML comienza con la etiqueta < html > Esta etiqueta indica el inicio del documento HTML y se encuentra en la primera línea del código HTML.

Encabezado: Dentro de la etiqueta HTML se encuentra el encabezado, que se define mediante la etiqueta < head > El encabezado contiene información sobre el documento HTML, como el título de la página, la descripción del contenido y otros metadatos que pueden ser útiles para los motores de búsqueda y otros sistemas.

Cuerpo: El cuerpo del documento HTML se define mediante la etiqueta < body > El cuerpo es donde se define el contenido principal de la página, como el texto, las imágenes, los enlaces y otros elementos.


Titulos


Las etiquetas < h1 > y se utilizan para definir un texto como título, indicando que es una cabecera (la h viene de header, cabecera en inglés) que queremos destacar sobre el resto del texto. Junto a < h1 > contamos con < h2 >, < h3 > y así hasta < h6 > para definir diferentes títulos, de mayor a menor importancia.

¿Que es CSS?


CSS (Cascading Style Sheets) es un lenguaje utilizado para describir la presentación visual de una página web escrita en HTML. Permite definir estilos y diseños para los elementos HTML, como el color, la fuente, el tamaño, la disposición y el espaciado, entre otros. CSS se utiliza para separar el contenido de una página web de su presentación, lo que permite una mayor flexibilidad y control en el diseño de la página. Los navegadores web interpretan el código CSS junto con el HTML para mostrar el contenido de la página con el estilo y diseño especificados.
